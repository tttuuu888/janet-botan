Reference

mpi/addcfunction
(mpi/add mpi x)

Add x to `mpi` and return the new MPI as a result. `x` can be either MPI or u32 number.

mpi/clear-bitcfunction
(mpi/clear-bit mpi bit)

Clears the specified bit of `n`, return self.

mpi/divcfunction
(mpi/div mpi1 mpi2)

Divide `mpi1` by `mpi2` and return quotient MPI and remainder MPI in tuple.

mpi/flip-signcfunction
(mpi/flip-sign mpi)

Flip the sign of `mpi` and return self.

mpi/from-hex-strcfunction
(mpi/from-hex-str value)

Create an MPI object with a hex string `value`.

mpi/from-intcfunction
(mpi/from-int value)

Create an MPI object with an integer `value`.

mpi/from-mpicfunction
(mpi/from-mpi mpi)

Create an MPI object with an MPI object `mpi`.

mpi/from-rngcfunction
(mpi/from-rng rng bits)

Create a `bits` size random MPI object with `rng`.

mpi/from-strcfunction
(mpi/from-str value)

Create an MPI object with an integer string `value`.

mpi/gcdcfunction
(mpi/gcd mpi other)

Return new MPI of the greatest common divisor of `mpi` and `other`.

mpi/get-bitcfunction
(mpi/get-bit mpi bit)

Returns 0 if the specified bit of n is not set, 1 if it is set.

mpi/inverse-modcfunction
(mpi/inverse-mod mpi modulus)

Return the inverse of MPI modulo `modulus`, or nil if no inverse exists.

mpi/is-negativecfunction
(mpi/is-negative mpi)

Return true if x is less than zero.

mpi/is-positivecfunction
(mpi/is-positive mpi)

Return true if x is greater than or equal to zero.

mpi/is-primecfunction
(mpi/is-prime mpi rng &opt prob)

Test if MPI is prime and return boolean. Default value of prob is 128.

mpi/is-zerocfunction
(mpi/is-zero mpi)

Return true if x is zero, otherwise false.

mpi/lshiftcfunction
(mpi/lshift mpi shift)

Left shift by specified bit count, return the result MPI.

mpi/mod-mulcfunction
(mpi/mod-mul mpi other modulus)

Return new MPI of the multiplication product of `mpi` and `other` modulo `modulus`.

mpi/mulcfunction
(mpi/sub mpi1 mpi2)

Multiply two MPI and return the new MPI as a result.

mpi/newcfunction
(mpi/new)

Create a new zero-valued MPI.

mpi/num-bytescfunction
(mpi/num-bytes mpi)

Return the number of significant bytes in the MPI.

mpi/pow-modcfunction
(mpi/pow-mod mpi exponent modulus)

Return new MPI to the `exponent` MPI power modulo `modulus` MPI.

mpi/rshiftcfunction
(mpi/rshift mpi shift)

Right shift by specified bit count, return the result MPI.

mpi/set-bitcfunction
(mpi/set-bit mpi bit)

Set the specified bit of `n`, return self.

mpi/subcfunction
(mpi/sub mpi x)

Subtract x from `mpi` and return the new MPI as a result. `x` can be either MPI or u32 number.

mpi/swapcfunction
(mpi/swap mpi1 mpi2)

Swap `mpi1` and `mpi2` values, return `mpi1`.

mpi/to-u32cfunction
(mpi/to-u32 mpi)

Convert the MPI to a uint32_t, if possible. Fails if MPI is negative or too large

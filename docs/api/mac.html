Reference

mac/clearcfunction
(mac/clear mac)

Reset the state of `mac` back to clean, as if no key and input has been supplied, return self.

mac/finalcfunction
(mac/final mac)

Finalize the MAC and return the output

mac/get-keyspeccfunction
(mac/get-keyspec mac)

Return the key spec of the `mac` in format of [max-key-length min-key-length mod-key-length].

mac/newcfunction
(mac/new name)

Creates a MAC of the given name, e.g., "HMAC(SHA-384)".

mac/output-lengthcfunction
(mac/output-length mac)

Return the output length of the `mac`

mac/set-keycfunction
(mac/set-key mac key)

Set the `key` for the MAC calculation, return self.

mac/set-noncecfunction
(mac/set-nonce mac key)

Set the `nonce` for the MAC calculation, return self.Note that not all MAC algorithms require a nonce. If a nonce is required, the function has to be called before the data is processed.

mac/updatecfunction
(mac/update mac input)

Add input to the MAC computation, return self.

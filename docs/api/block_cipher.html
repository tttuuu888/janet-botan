Block Ciphers

Random Number Generators Hash Functions

Index

block-cipher/block-size block-cipher/clear block-cipher/decrypt block-cipher/encrypt block-cipher/get-keyspec block-cipher/name block-cipher/new block-cipher/set-key

Reference

block-cipher/block-sizecfunction
(block-cipher/block-size bc)

Return the block size of this cipher.

block-cipher/clearcfunction
(block-cipher/clear bc)

Clear the internal state (such as keys) of this cipher object, but do not deallocate it.

block-cipher/decryptcfunction
(block-cipher/decrypt bc input)

Decrypt `input` data. The key must have been set beforehand.

block-cipher/encryptcfunction
(block-cipher/encrypt bc input)

Encrypt `input` data. The key must have been set beforehand.

block-cipher/get-keyspeccfunction
(block-cipher/get-keyspec mac)

Return the key spec of this cipher in format of [max-key-length min-key-length mod-key-length].

block-cipher/namecfunction
(block-cipher/name bc)

Return the name of this block cipher algorithm, which may nor may not exactly match what was passed to `block-cipher/init`.

block-cipher/newcfunction
(block-cipher/new name)

Create a new cipher mode object, `name` should be for example "AES-128" or "Threefish-512"

block-cipher/set-keycfunction
(block-cipher/set-key bc key)

Set the cipher key, which is required before encrypting or decrypting.
